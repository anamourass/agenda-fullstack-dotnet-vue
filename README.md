# 📇 Agenda Full Stack (.NET 10 & Vue 3)

Projeto desenvolvido para demonstrar competências em desenvolvimento Full Stack, utilizando as tecnologias mais recentes do ecossistema Microsoft e JavaScript.

## 🚀 Tecnologias Utilizadas

### Backend (C# / .NET 10)
- **ASP.NET Core Web API**
- **Entity Framework Core** (Banco em Memória)
- **AutoMapper** (Mapeamento de DTOs)
- **FluentValidation** (Regras de Negócio)
- **xUnit & Moq** (Testes Unitários com 100% de cobertura no Service)
- **Swagger** (Documentação da API)

### Frontend (Vue.js 3)
- **Composition API**
- **Axios** (Integração com API)
- **Componentização** Reutilizável

## 🛠️ Como Executar o Projeto

1. **Backend:** - `cd AgendaApiNova`
   - `dotnet run`
2. **Frontend:**
   - `cd agenda-vue`
   - `npm install`
   - `npm run dev`
